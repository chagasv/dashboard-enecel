Modo de Leitura Online

Por razões de segurança, limites de recursos e proteção de rede das APIs (CCEE, ONS, BBCE), as rotinas de ETL e atualizações diretas de dados via web foram movidas exclusivamente para o ambiente local.

Como atualizar os dados da base:
  1. Abra a pasta do projeto no seu computador local.
  2. Execute o arquivo atualizar_local.bat com clique duplo (ou rode python atualizar_local.py no terminal).
  3. Escolha a base de dados que deseja atualizar na interface interativa do terminal.
  4. O script rodará localmente, consolidará os dados das planilhas, atualizará os caches locais e fará o envio (git push) de forma 100% automatizada.
  5. O servidor na nuvem detectará a atualização no GitHub e atualizará os dados deste dashboard em poucos instantes.
